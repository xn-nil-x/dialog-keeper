## Contributors

We welcome contributions from the community! This project follows the [Contributor Covenant Code of Conduct](CODE_OF_CONDUCT.md).

### How to become a contributor

1. Fork the repository
2. Create a feature branch
3. Make your changes following our [Coding Standards](docs/development/CODING_STANDARDS.md)
4. Submit a Pull Request
5. Your name will be added here after your first merged PR!
